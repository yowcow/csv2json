CSV2JSON
========

Reads CSV input and writes JSON output

HOW TO USE
----------

    csv2json [-i <input CSV file>] [-o <output JSON file>]


* `-i <input file>`: CSV input (default: STDIN)
* `-o <output file>`: JSON output (default: STDOUT)

HOW TO INSTALL
--------------

    go get -u -v github.com/yowcow/csv2json
